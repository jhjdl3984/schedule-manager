# 🗓️ Schedule Manager (CLI)

간단한 일정 관리 프로그램입니다.  
사용자는 터미널에서 일정을 추가하고, 조회하고, 완료 처리할 수 있습니다.  
Python과 MySQL(Pymysql)을 사용하여 구성되었습니다.

---

## 📂 기능 소개

### 1️⃣ 일정 추가 (Insert)
- 제목, 설명, 시작일, 종료일을 입력받아 DB에 저장합니다.  
- 날짜 형식: `YYYYMMDDHHMMSS`

### 2️⃣ 일정 보기 (Select)
- `schedules` 테이블의 모든 일정을 조회합니다.  
- 각 일정 표시 항목:
  - ID
  - 제목
  - 설명
  - 시작 날짜
  - 종료 날짜
  - 완료 여부 (`완료` / `미완료`)

### 3️⃣ 일정 완료 처리 (Update)
- ID 값을 입력받아 해당 일정을 `is_completed = TRUE` 처리합니다.

### 4️⃣ 종료
- 프로그램을 종료합니다.
